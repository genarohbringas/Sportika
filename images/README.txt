Para usar tus propias fotos locales:
1. Renombra tus archivos a: padel-1.jpg, padel-2.jpg, padel-3.jpg, cafeteria.jpg
2. Si Windows muestra "padel-1.jpg" pero el tipo es "JPEG", el archivo real puede ser "padel-1.jpg.jpeg"
   - En el Explorador: Ver > Mostrar > Extensiones de nombre de archivo
   - Renombra eliminando el .jpeg extra para dejar solo .jpg
3. Actualiza las rutas en index.html cambiando las URLs de Pexels por: images/padel-1.jpg, etc.
